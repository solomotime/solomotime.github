Technology behind the blog

Hacker Labs is powered by OctoPress and hosted on github as a static page. The theme is based on svbtle (created by Dustin Curtis). Svbtle theme was ported to WordPress by Gravity On Mars, I hacked it to work with OctoPress and made it more suitable for me. You can find my version of this theme on GitHub.

Since this site is just a bunch of static pages with no server-side generated code, I have to use external services for all the dynamic features. I use Swiftype to power the site search, wufoo for the contact form and Disqus for the comments.
